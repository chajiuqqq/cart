# My cart 
该项目是基于JavaEE的web项目，实现了基本的购物功能，并保存用户数据和订单到MySQL数据库。功能包括了`用户注册`，`用户登录`，`商品页面`，`购物车页面`，`用户订单中心页面`。

该项目使用了tomcat服务器，servlet技术，jsp技术，同时包括基本的前端技术：html,css,js,Ajax,jQuery。有部分实现了mvc设计模式。
